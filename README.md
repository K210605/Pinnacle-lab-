## 🔐 Text Encryption Tool

This project implements a text encryption and decryption system using Fernet (AES-based encryption).

### Features:
- Encrypt plain text
- Decrypt encrypted text
- Secure key-based encryption

### How to Run:
pip install -r text-encryption/requirements.txt
python text-encryption/main.py
